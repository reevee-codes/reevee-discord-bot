# Discord Bot Python

A modular Discord bot written in Python.

## Features
- Easily extensible command system
- Ready-to-use commands:  
  - `ping`
  - `echo`
  - `fact`
  - `quote`
  - `chat`

## Setup

1. **Clone the repository**
2. **Install dependencies**  
   ```
   pip install -r requirements.txt
   ```
3. **Configure your bot token** (see `app/bot.py`)
4. **Set your OpenAI API key (needed for AI commands)**  
   For example (Windows Command Prompt):
   ```
   set OPENAI_API_KEY=your-api-key-here
   ```
   Or set in your config if applicable.
5. **Run the bot**  
   ```
   python app/bot.py
   ```
